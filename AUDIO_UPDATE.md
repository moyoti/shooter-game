# 🎵 音效更新说明

## 新增功能

### 🔊 音效系统
使用 Web Audio API 实时生成音效，无需外部文件：

1. **射击音效** - 每次射击时播放
2. **爆炸音效** - 敌人死亡时播放
3. **升级音效** - 玩家升级时播放
4. **连击音效** - 每 10 连击播放特殊音效

### 🎵 背景音乐 (BGM)
- 使用之前 AI 生成的音乐作为 BGM
- 自动循环播放
- 音量适中，不影响游戏体验

### 🎛️ 音频控制
游戏右上角新增两个控制按钮：
- 🔊 **音效开关** - 控制射击、爆炸等音效
- 🎵 **BGM 开关** - 控制背景音乐

## 使用方法

1. 打开游戏：https://moyoti.github.io/shooter-game/
2. 点击"开始游戏"
3. 音频系统会自动初始化
4. 右上角按钮可随时开关音效

## 技术实现

- **Web Audio API** - 实时生成音效，无需加载外部文件
- **HTML5 Audio** - 播放背景音乐
- **Phaser 3** - 游戏引擎集成
